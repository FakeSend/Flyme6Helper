说明

本模块是将「Flyme6助手」这个模块里面的「和谐主题」部分的代码抽了出来单独做了一个模块。

为什么要单独抽出来？

因为原先的模块在 Android 6.0 以上(SDK >= 23)会出 bug，会提示「不受支持的安卓版本」，导致还没进去就闪退了。

原模块只有在你进模块里面后打开一个开关才能用，因为进不去所以也就用不了了，为了避免这种情况，所以本模块不提供开关控制功能。

原模块里的其他功能比如「净化广告」、「Host设置」、「应用管理」等功能和「和谐主题」来比，明显很多人是冲着这个功能来的， 如果只能选择一个功能，反正我是毫不犹豫选择「和谐主题」，因为其他功能其他 APP 或模块都能完全代替他们，甚至比他们做的更好。

原作者最后适配到了当时最新的「主题美化」版本 6.12.1，然而今天我接手的时候已经更新到了 6.13.1，原作者曾经说过不适配了，但是诈尸复活了一次， 然后又说「插件没啥人用，自己也没魅族的机子，而且魅族作死取消root。 插件再更新也没啥意思了。」，看样子是又不打算适配了。

就在我接手的时候，去 GitHub 上得到源码后就来适配了，我一开始没发现他只是更新了应用，但是并没有更新 GitHub 上的源码，经过我一系列的摸索之后， 现在应该算是没 bug 了，如果还有 bug 请告诉我。

此次适配是参考「主题美化」版本 6.13.1 的源码适配的，删除了一些旧版本的 Hook 代码，所以不敢保证以前的版本是否能用，如果你是 6.0 以下的系统可以去原插件博客里找， 那里面的最高可以支持到 6.12.1，如果还是有 bug 或不兼容，只要能试用主题，试用后再冻结「主题美化」就行了，这是万不得已的情况下才使用的也是最暴力的破解方法。
